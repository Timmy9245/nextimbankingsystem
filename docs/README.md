# Docs

Reference documentation for the NexTim banking project.

| File                  | Purpose                                                |
| --------------------- | ------------------------------------------------------ |
| `ERD.md`              | Entity-relationship diagram (text/markdown)            |
| `normalization.md`    | 1NF → 3NF walkthrough for the schema                   |
| `project_report.md`   | Project report: goals, stack, features, security       |

The original layout requested `ERD.png`, `normalization.docx`, and
`project_report.docx`. Binary `.png` / `.docx` files cannot be authored
from inside the codebase, so the same content is provided as Markdown
here — open in any editor and export to PNG/DOCX as needed.
